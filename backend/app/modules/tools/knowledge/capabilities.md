# Что умеет FinAgent

FinAgent — локальный агент-аналитик банковских выписок. Он работает полностью на машине
пользователя: данные (выписки, транзакции, история чата) никуда не отправляются, кроме
запросов к выбранному LLM-провайдеру (OpenAI или локальный Ollama).

Агент умеет:
- Отвечать на вопросы о тратах на естественном языке, обращаясь к реальным данным через
  read-only SQL-запросы (инструмент `sql_query`) — никогда не выдумывает суммы.
- Строить графики (столбцы, линии, кольцевые диаграммы) по агрегированным данным.
- Сравнивать два периода между собой: суммы по категориям, дельты, рост в процентах и главный
  драйвер изменения.
- Сам определять, к какой выписке или периоду относится вопрос, если пользователь не указал
  файл явно — а если однозначно понять нельзя, переспрашивает вместо того чтобы гадать.
- Категоризировать траты по продавцам (правила + LLM для новых продавцов).
- Разбирать выписки в форматах CSV и PDF (включая OCR для сканов).

Агент **не даёт** финансовых инвестиционных советов — только аналитику прошлых трат — и не
отвечает на вопросы, не связанные с финансами пользователя.
